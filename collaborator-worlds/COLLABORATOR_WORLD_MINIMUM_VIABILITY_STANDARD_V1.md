# COLLABORATOR_WORLD_MINIMUM_VIABILITY_STANDARD_V1

## Status
Active standard.

## Purpose
Set a minimum quality floor for collaborator worlds so scale does not produce generic, weak, or under-authored lanes.

## Core Principle
A collaborator world is not properly opened just because a thread exists.

## Minimum viability requirements
- role clarity
- route clarity
- chamber-opening quality
- Citadel soul presence
- practical invitation into use
- enough specificity that the lane feels authored for the person

## Failure sign
If a world exists but feels generic, low-signal, or emotionally flat, it has not met minimum viability.

## Final Sentence
A collaborator world should open as a real chamber, not as a generic thread.
